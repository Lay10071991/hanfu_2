# 汉服文化交流平台 - 项目说明文档

## 📋 项目概述

汉服文化交流平台是一个集汉服知识学习、文化传播、社区互动、活动召集、店铺测评于一体的综合性Web应用平台。旨在传承和弘扬中华传统服饰文化，为汉服爱好者提供一个学习交流的平台。

### 项目信息
- **项目名称**: 汉服文化交流平台
- **技术栈**: Spring Boot + Vue 3 + Element Plus
- **开发时间**: 2023-2024
- **项目类型**: 全栈Web应用

---

### 1. 图片上传功能
#### 实现方式
- **后端**: FileUploadController 处理文件上传
- **存储路径**: `E:/xwechat_files/.../uploads/`
- **访问URL**: `http://localhost:8082/uploads/文件名`
- **文件命名**: UUID + 原始扩展名
- **文件限制**: 图片格式，最大5MB

#### 前端实现
```javascript
// 使用FormData上传
const formData = new FormData()
formData.append('file', file)

const response = await fetch('http://localhost:8082/api/upload/image', {
  method: 'POST',
  body: formData
})
```

#### 管理后台上传组件
- 点击上传区域选择文件
- 实时图片预览
- 删除已上传图片
- 上传进度提示

### 2. 图片URL处理
#### imageHelper.js 工具函数
```javascript
// 统一处理图片URL
export function getImageUrl(url) {
  if (!url) return ''
  if (url.startsWith('http://') || url.startsWith('https://')) {
    return url
  }
  return `http://localhost:8082${url.startsWith('/') ? '' : '/'}${url}`
}
```

#### 1. 环境要求
- JDK 17+

- Maven 3.6+

#### 2. 配置数据库
```properties
# application.properties
spring.datasource.url=jdbc:mysql://localhost:3306/hanfu_platform?useUnicode=true&characterEncoding=utf8
spring.datasource.username=root
spring.datasource.password=your_password
```

#### 3. 创建上传目录
```bash
# 创建uploads目录
mkdir -p /path/to/uploads
```

#### 4. 修改上传路径
```java
// FileUploadController.java
private static final String UPLOAD_DIR = "/path/to/uploads";
```

#### 5. 运行项目
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

#### 1. 环境要求
- Node.js 16+

#### 2. 安装依赖
```bash
cd frontend
```

```bash
npm run dev
```

#### 4. 生产构建

